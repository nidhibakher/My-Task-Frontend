HTML Structure Explanation

  • <!DOCTYPE html>: Declares the document as HTML5.
  • <html>: The root element that wraps all content on the page.
  • <head>: Contains meta-information, links, and scripts for the page.
  • <meta charset='utf-8'>: Sets character encoding to UTF-8 for wide language support.
  • <meta http-equiv='X-UA-Compatible' content='IE=edge'>: Ensures compatibility with the latest Internet Explorer rendering mode.
  • <title>: Sets the page title shown in the browser tab.
  • <meta name='viewport' content='width=device-width, initial-scale=1'>: Makes the page responsive on all devices.
  • <link rel='stylesheet' type='text/css' media='screen' href='main.css'>: Links an external CSS file for styling.
  • <script src='main.js'></script>: Links an external JavaScript file for interactivity.
  • <body>: Contains all the visible content of the web page.
  • </body> & </html>: Closing tags for the body and html elements.
