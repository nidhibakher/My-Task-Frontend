Paragraph-Tag

  • <p> tag is used to define a paragraph in HTML.
  • It automatically adds space before and after the paragraph for better readability.
  • Paragraphs help organize text content on a webpage.
  • You can use multiple <p> tags to create multiple paragraphs.
  • It is a block-level element, so it starts on a new line.
